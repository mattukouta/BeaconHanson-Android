# FunLocks-hanson-android
FunLocksの「Beacon×Android」勉強会用のサンプルアプリ

## サンプルアプリ概要
[AltBeacon](https://github.com/AltBeacon/android-beacon-library)を使用してiBeaconを検出し，簡易的なリストでBeacon情報を表示させるアプリです．
